# AI Engineer Classroom App

An interactive self-paced classroom application based on the "AI Engineer Roadmap: Zero to Hired in 6 Months". 

## Features
- **Self-Paced Learning:** All modules, weeks, and topics are accessible from day one.
- **Progress Tracking:** Track which topics are completed, what the current lesson is, and overall progress.

## Project Structure
- `frontend/`: Next.js application representing the interactive user interface.
- `backend/`: FastAPI backend built in Python, managing curriculum data and user progress with SQLite.
